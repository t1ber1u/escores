Placement of tsconfig.scripts.json and tsconfig.tests.json

Both tsconfig.scripts.json and tsconfig.tests.json should be placed in the root directory:

	•	tsconfig.scripts.json: Specific to the scripts folder.
	•	tsconfig.tests.json: Specific to the tests in the src folder and the WebDriverIO configurations.

Steps to Create and Use the Files

	1.	Create tsconfig.scripts.json:
Place this file in the root directory:

{
  "extends": "./tsconfig.json",
  "compilerOptions": {
    "types": ["node"]
  },
  "include": ["scripts/**/*"]
}
------------------
tsconfig.tests.json

{
  "extends": "./tsconfig.json",
  "compilerOptions": {
    "types": ["jest", "node", "@wdio/globals/types", "expect-webdriverio", "@wdio/cucumber-framework"]
  },
  "include": ["src/**/*", "wdio.conf.ts", "wdio.conf.gitlab.ts"]
}-------------------
